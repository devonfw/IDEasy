Automated dev environment setup for any project.

  • Install IDEasy

    macOS / Linux

    bash -c "$(curl -fsSL https://raw.githubusercontent.com/devonfw/IDEasy/main/install.sh)"

    Windows (PowerShell)

    irm https://raw.githubusercontent.com/devonfw/IDEasy/main/install.ps1 | iex

    Paste the command for your platform into a terminal. The script detects your OS and architecture, downloads the latest release, and runs setup automatically.

    Requires git. macOS and Linux also need curl. Works on macOS (Intel & Apple Silicon), Linux (x64 & ARM64), and Windows (x64).

  • What is IDEasy?

    IDEasy automates the setup and updates of development environments. Create a small git repo with your project's dev config. Every developer runs one command and gets all tools in the right versions — no manual steps, no "it works on my machine."

    $ ide create my-project
Setting up project my-project...
Installing Java 21.0.4...
Installing Maven 3.9.9...
Project my-project is ready.

  • IDEasy keeps tools sandboxed inside your project. Each project has its own Java, Node, Maven — whatever it needs. Multiple projects with different tool versions coexist without conflict. Your OS stays clean.

    $ cd cool-project && ide
Environment set for ~/projects/cool-project
$ mvn -v
Apache Maven 3.9.9
Java version: 21.0.4

$ cd ../legacy-app && ide
Environment set for ~/projects/legacy-app
$ mvn -v
Apache Maven 3.6.0
Java version: 11.0.2

  • Project settings live in a git repository your team shares. Code formatters, tool versions, IDE plugins — all configured once and distributed to every developer. Changes go through pull requests. No more diff-wars.

    my-project-settings/
├── ide/
│   ├── java/
│   │   └── version      # 21.0.4
│   ├── mvn/
│   │   └── version      # 3.9.9
│   └── vscode/
│       └── plugins.json
└── workspace/
    └── formatter.xml

  • Launch your IDE preconfigured with the right plugins, formatters, and workspace settings. Supports Eclipse, IntelliJ IDEA, VS Code, and Android Studio.

    $ ide intellij
Launching IntelliJ for workspace main...

$ ide vscode
Launching VS Code for workspace main...

$ ide eclipse
Launching Eclipse for workspace main...

  • Works on Windows (PowerShell), macOS (Intel & Apple Silicon), and Linux (x64 & ARM64). Same workflow everywhere. Supports Bash, Zsh, PowerShell, Windows Terminal, and Git Bash.

  • Uninstalling is clean and complete. IDEasy is non-invasive — nothing is installed globally on your system. All tools stay local to your project folder.

    $ ide uninstall
IDEasy has been removed.
