Automated dev environment setup for any project.

  • 1. Install IDEasy

    bash -c "$(curl -fsSL https://raw.githubusercontent.com/devonfw/IDEasy/main/install.sh)"

    Run this command in any terminal. Works on macOS, Linux, and Windows. The script detects your OS and architecture, downloads the latest release, and runs setup automatically.

    Requires git and curl. On Windows, use Git Bash (included with Git for Windows).

  • 2. Set Up Your Project

    ide create my-project

    After restarting your terminal, create a project. You will be prompted for your team's settings repository URL, or press Enter to use default settings. IDEasy installs all required tools in the right versions automatically.

    $ ide create my-project
Setting up project my-project...
Installing Java 21.0.4...
Installing Maven 3.9.9...
Project my-project is ready.

  • What is IDEasy?

    IDEasy automates the setup and updates of development environments. Create a small git repo with your project's dev config. Every developer runs one command and gets all tools in the right versions — fully automated and reproducible.

    my-project-settings/
├── ide/
│   ├── java/
│   │   └── version      # 21.0.4
│   ├── mvn/
│   │   └── version      # 3.9.9
│   └── vscode/
│       └── plugins.json
└── workspace/
    └── formatter.xml

  • IDEasy keeps tools sandboxed inside your project. Each project has its own Java, Node, Maven — whatever it needs. Multiple projects with different tool versions coexist without conflict. Your OS stays clean.

    $ cd cool-project && ide
Environment set for ~/projects/cool-project
$ mvn -v
Apache Maven 3.9.9
Java version: 21.0.4

$ cd ../legacy-app && ide
Environment set for ~/projects/legacy-app
$ mvn -v
Apache Maven 3.6.0
Java version: 11.0.2

  • Project settings live in a git repository your team shares. Tool versions, IDE plugins, code formatters — all configured once and distributed to every developer. Changes go through pull requests. No more diff-wars.

    $ ide update
Pulling latest settings...
Updated Java from 21.0.3 to 21.0.4
Updated Maven from 3.9.8 to 3.9.9
Settings are up to date.

  • Launch your IDE preconfigured with the right plugins, formatters, and workspace settings. Supports Eclipse, IntelliJ IDEA, VS Code, and Android Studio.

    $ ide intellij
Launching IntelliJ for workspace main...

$ ide vscode
Launching VS Code for workspace main...

$ ide eclipse
Launching Eclipse for workspace main...

  • Works on Windows (PowerShell), macOS (Intel & Apple Silicon), and Linux (x64 & ARM64). Same workflow everywhere. Supports Bash, Zsh, PowerShell, Windows Terminal, and Git Bash.

  • Uninstalling is clean and complete. IDEasy is non-invasive — nothing is installed globally on your system. All tools stay local to your project folder.

    $ ide uninstall
IDEasy has been removed.
